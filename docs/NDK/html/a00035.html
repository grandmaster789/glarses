T5_EXPORT T5_Result t5ReleaseGlasses ( T5_Glasses  glasses)

Release previously-acquired glasses.

Release glasses that were previously acquired for exclusive operations by the client. After calling this, exclusive operations cannot be used with the glasses unless the glasses are again acquired and readied.

Threading
Exclusivity group 1 - Functions in this group must not be called simultaneously from different threads.
Parameters
[in]glasses- T5_Glasses returned by T5ApiSys::createGlasses()
Return values
T5_SUCCESSGlasses released.
